## 💡 Philosophy

Each folder in this repo is a **self-contained experiment** — its own README, its own context, its own conclusions. Nothing here is "production-ready" in the enterprise sense. Everything here has been **run against real problems**.

The guiding principle comes from Bungay:

> *"Do not add unnecessary detail. Specify the result to be achieved, not how to achieve it."*

That applies equally to prompts, agent instructions, and documentation.

---

## 🚀 Getting started

Browse a folder that interests you. Read its `README.md` first — each one explains what problem it solves and how to use it. There is no global install step.

## 🤝 Contributing
